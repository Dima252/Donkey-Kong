# Donkey Kong - C++ OOP Implementation

This project is a C++ implementation of the classic arcade game **Donkey Kong**, developed by Shalev Yosefashvili and Dimitry Todoseyev. Its main goal is to demonstrate robust Object-Oriented Programming (OOP) principles in C++ through game mechanics.

## Prerequisites
To compile and run this project in its original format, you will need:
- Windows OS
- **Visual Studio 2022** (or compatible versions) with the **Desktop development with C++** workload installed.
- The platform toolset installed (the original project file targets `v143`).

## How to Build the Game
### Option 1: Using Visual Studio (Recommended)
1. Open the sln file in Visual Studio.
2. Ensure the build configuration drop-down at the top is set to **Debug** and **x64**.
3. Right-click the `donkey_kong` project in the Solution Explorer and click **Build**, or simply press `Ctrl+Shift+B`.
